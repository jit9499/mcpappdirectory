About MCP App Directory

MCP App Directory is the first AI-graded quality directory for MCP (Model Context Protocol) servers. We index every public MCP server, score it on 8 quality parameters, and present an objective 0-100 score so you can find the best tools for your AI agents.

MCP servers are the bridge between AI agents and the outside world — APIs, databases, files, search, and more. With thousands available, finding a quality server is hard. We solve that.

What We Score

Every server is analyzed on 8 parameters using real GitHub data:

  • Recency (20pts) — Last commit within 90 days
  • Documentation (15pts) — Quality README
  • Tests (10pts) — Test infrastructure
  • Auth/Security (10pts) — Auth patterns detected
  • Star Velocity (15pts) — Stars per month
  • Security Audit (15pts) — No obvious security issues
  • Install Docs (15pts) — Clear setup + usage examples
  • Activity Bonus (5pts) — Recent commits

Grade Scale

  • A (90-100) — Top Quality. Production-ready.
  • B (70-89) — Good Quality. Well-maintained.
  • C (50-69) — Average. Usable but may lack polish.
  • D (30-49) — Below Average. Limited maintenance.
  • F (0-29) — Low Quality. Minimal or abandoned.

Built With

We use the GitHub API to fetch real repository data (stars, forks, commits, README quality, test files) and compute scores algorithmically. No AI hallucination — just real data + transparent scoring.
